# KuaforRandevuWebSitesi
[![](https://img.shields.io/badge/WEB_SİTESİ-Giriş_Yap-blue?style=for-the-badge&logo=firebase)](https://karizmaerkekkuaforu.netlify.app)
Karizma Erkek Kuaförü için geliştirilmiş, kullanıcıların çevrim içi randevu oluşturabildiği bir web tabanlı randevu yönetim sistemidir. Proje, modern web teknolojileri kullanılarak geliştirilmiş olup randevu verilerinin saklanması ve yönetimi için Firebase Database altyapısı kullanılmıştır.

Proje Özeti

Bu web sitesi üzerinden kullanıcılar kuaför için randevu oluşturabilir. Sistem, randevu verilerini Firebase veritabanına kaydeder ve yönetici paneli üzerinden randevu kayıtlarının görüntülenmesi, eklenmesi ve silinmesi işlemleri yapılabilir.

Site Bölümleri

Ana Sayfa
Hakkında Sayfası
Fiyat Listesi Sayfası
Randevu Oluşturma Sayfası
Yönetici Paneli

Özellikler

Online randevu oluşturma
Randevu bilgilerinin Firebase Database’e kaydedilmesi
Mevcut randevuları görüntüleme
Randevu kaydı ekleme ve silme
Yönetici paneli üzerinden randevu yönetimi
Çok sayfalı web site yapısı

Kullanılan Teknolojiler

HTML5
CSS
JavaScript
Firebase Realtime Database

Amaç

Bu proje, gerçek bir işletme senaryosu için randevu alma ve yönetme sürecini dijital ortama taşımayı hedefleyen bir web uygulamasıdır.
